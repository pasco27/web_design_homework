Summary: Lat-titude with Attitude

The premise of this project was to create a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. To accomplish this, a Python library was utilized in conjuncttion with the OpenWeatherAPI to create a representative model of weather across world cities. This page was created as a fun visual summary of sorts. The first requirement of the WeatherPy project was to create a series of scatter plots to showcase the following relationships:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
