Summary: Lat-titude with Attitude

WeatherPy was a project that was assigned within the Rice University Data Visualization and Analytics Bootcamp for the purposes of displaying newly gained knowledge of API's and their usage. Given the opportunity to show knowledge of the application programming interface and the vast expanse of their utilization was an exciting and challenging task. The premise of the WeatherPy project was to create a Python script to visualize the weather of 500+ cities across the world of varying distances from the equator. To accomplish this, a Python library was implemented in conjuncttion with the OpenWeatherAPI to create a representative model of weather across world cities. This page was created as a fun visual summary of sorts, displaying the resulting Matplotlib files that came of the project. The first requirement of the WeatherPy project was to create a series of scatter plots that showcased the following relationships:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude

In similar fashion, this website was created in an effort to tinker with newly forming knowledge of HTML/CSS/Bootstrap tools. The pages available here were formed to show the intricate details of each plot, and give marginally adequate explainations as to their meaning.
